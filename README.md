# Capitalizer

Use the included en_US.aff and en_US.dic dictionary files and the below prompt to create a simple capitalization algorithm in Python with unit testing to confirm reliable output.

## Prompt

Create a Python command-line application that capitalizes brand/domain names by detecting English dictionary words within them, using dictionary files provided in the root directory as en_US.aff and en_US.dic. Accept space-separated names as command-line arguments and output the capitalized versions.

Write a pytest test suite that validates the functionality against the examples below. Use these failing tests to drive the implementation and create algorithm rules as necessary to deliver correct results.
- quicksilver -> Quicksilver
- silverlakedesign -> SilverLakeDesign
- darkstoneforge.com -> DarkStoneForge.com
- northwindtravel.org -> NorthWindTravel.org
- notepilot -> NotePilot
- meetingflow -> MeetingFlow
- packora -> Packora
- cleancarton -> CleanCarton
- savvio -> Savvio
- brightbalance -> BrightBalance
- campaignbuilder -> CampaignBuilder

## Example

There is an example Python project created from the output of this prompt in the "example" branch in GitHub. This has been left out of the main branch to avoid any contamination when an agent goes to create a new project.
